# Varhaiskasvatuksen palveluohjaus

## Yleistä

Palveluohjauksesta lapsen huoltaja saa opastusta kunnan varhaiskasvatuksen paikoista, sisällöstä ja toimintaohjeita palvelun saamiseksi.

## Tarvittavat esitiedot

- Lapsen ikä
- Lapsen asuinalue kunnassa
- Lapselle tarvittava hoitoaika

## Mitä asiointikanavia voit käyttää

Voit olla yhteydessä palveluohjaukseen seuraavissa asiointikanavissa:

- Soittamalla palveluohjauksen puhelinasiointiin
- Lähettämällä viestin palveluohjauksen sähköpostiosoitteeseen
- Lähettämällä viestin palveluohjauksen postiosoitteseen

## Toimintaohjeet

1. Ota yhteyttä palveluohjaukseen asiointikanavan kautta.
2. Ilmoita tarvittavat esitiedot palveluneuvojalle.
3. Toimi saamasi ohjauksen mukaisesti.
